# Platform Formatting Guide

> Shared reference for message formatting across platforms

## Discord

- **No markdown tables** — Use bullet lists instead
- **Links** — Wrap multiple links in `<>` to suppress embeds: `<https://example.com>`
- **Code blocks** — Use triple backticks with language

## Telegram

- **Headers** — Use **bold** or CAPS for emphasis
- **Tables** — Limited support, prefer lists
- **Links** — Standard markdown works

## WhatsApp

- **No headers** — Use **bold** or CAPS for emphasis
- **Lists** — Use simple dashes or numbers
- **Code** — Use single backticks inline

## Voice (TTS)

- Use voice for stories, summaries, "storytime" moments
- More engaging than walls of text
- Pick appropriate voices for characters
